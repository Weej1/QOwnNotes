# Icons

To generate icons for the whole applications, run the following command:

```bash
# Generate all icons
./generate-icons.sh
```

It uses `icon.svg` and `icon-dark.svg` as the source files and generates the
icons in the `icons` folder, for `src/images` and the snaps.
